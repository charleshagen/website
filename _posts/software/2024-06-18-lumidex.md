[Lumidex](https://github.com/alexhelms/lumidex/releases/latest), created by [Alex Helms](https://alexhelms.com/), is a standalone application for Windows, MacOS and Linux built for organizing and visualizing astronomical data. Lumidex provides a local SQLite database for accessing and sorting through XISF and FITS files and their associated metadata without altering your files. Additionally, statistics and data can be exported in markdown table or Astrobin formats, reducing the time and complexity of accounting for your data.

# Installation
{:style="font-size: 2em; padding-bottom: 1rem;"}

## <img src="/assets/img/windows_logo.png" alt="Windows Logo" style="height: 1.2em; vertical-align: middle; padding-right: 0.3em"> Windows
- Download the [latest version of Lumidex](https://github.com/alexhelms/lumidex/releases/latest){:target="_blank"} for Windows.
- Run the installer and follow the on-screen instructions.

## <img src="/assets/img/apple_logo.png" alt="Apple Logo" style="height: 1.2em; vertical-align: text-top; padding-right: 0.3em;"> MacOS
- Download the [latest version of Lumidex](https://github.com/alexhelms/lumidex/releases/latest){:target="_blank"} for MacOS.
- Open Lumidex. 
  - MacOS will stop it from launching and give you warning that it could not verify the app. Dismiss the warning by selecting **'Done'**. 
  - Open System Settings, go to Privacy & Security, scroll down to the bottom and select **'Open Anyway'** for `Lumidex.app`. 
  - A dialog will open, select **'Open Anyway'** and login to confirm. **This must be repeated for every update of Lumidex.**

## <img src="/assets/img/ubuntu_logo.png" alt="Ubuntu Logo" style="height: 1.2em; vertical-align: middle; padding-right: 0.3em"> Linux (Ubuntu)
~~~bash 
#file: "bash" 
sudo apt-get install dotnet-sdk-9.0 appmenu-registrar
git clone https://github.com/alexhelms/lumidex.git
cd lumidex
dotnet restore
dotnet build
dotnet run --project Lumidex.Desktop
~~~

# Features
{:style="font-size: 2em; padding-bottom: 1rem;"}

- Windows and MacOS support
    - Debian-based linux distros should also work but no `dpkg` is available
- `.FITS` and `.XISF` file support
- Search filters
    - Object name
    - File path
    - Date range
    - Exposure
    - Filter
    - Image type 
    - And more!
- Visualizations 
    - Integration over time bar chart
    - Integration per target pie chart
    - Integration over time heatmap
- Recursive scanning
    - Quick scan
    - Duplicate scrubbing
- Tagging
- Export data
- Object aliases
- Database editing
- Exposure summary
    - Copy as text
    - Copy as markdown
- Astrobin CSV export
    - Astrobin filter IDs
- Multiple libraries
- Duplicate detection
- Open in PixInsight
